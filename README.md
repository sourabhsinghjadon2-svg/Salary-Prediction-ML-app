# 💰 AI Salary Prediction App

## 📌 Overview
This is a Machine Learning web app that predicts salary based on user inputs like experience, education, job role, and age.

## 🚀 Features
- Real-time salary prediction
- Interactive UI using Streamlit
- ML model trained on real dataset
- Beautiful dashboard with metrics

## 🛠 Tech Stack
- Python
- Streamlit
- Scikit-learn
- XGBoost
- Pandas
- Joblib

## 📊 Model
Trained using regression models and best model selected based on R² score.

## ▶ How to Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
